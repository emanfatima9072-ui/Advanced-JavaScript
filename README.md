# Advanced JavaScript Notes (Chai Aur Code)
A comprehensive beginner-friendly code-along repository created while following the Chai Aur Code JavaScript Playlist (Videos 36–51).
This repository contains detailed notes, commented examples, and practical code snippets that explain how JavaScript works internally instead of only focusing on syntax.

- Iterators
